How to run the Society Helpdesk (SH) Project on your local pc:

(Prerequsite php development environment)

1. Extract the rar file and copy SH folder

2. Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

3. Open PHPMyAdmin (http://localhost/phpmyadmin)

4. Create a database with name sh

5. Import sh.sql file(given inside the rar package)

6. Copy and replace the config.php file in  "SH/users/includes/" and "SH/admin/include/" folder 

7. Open your browser and goto : http://localhost/sh

Note: For admin panel 

http://localhost/sh/admin (admin panel)
